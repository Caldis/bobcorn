Installation manuelle CLI

Installation manuelle et configuration PATH pour Bobcorn CLI.

Configuration manuelle

Si vous preferez ne pas utiliser le panneau Parametres, vous pouvez configurer le CLI manuellement.

Option A : Auto-installation depuis le binaire CLI

Localisez le binaire CLI dans le bundle de l'application Bobcorn et executez directement la commande install :

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

Cela enregistre automatiquement la commande bobcorn dans votre PATH.

Option B : Ajouter au PATH manuellement

Creez un script wrapper qui invoque le binaire CLI avec Node.js :

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

Assurez-vous que le repertoire du wrapper est dans votre PATH. Sur macOS/Linux, ajoutez ~/.local/bin a votre profil shell. Sur Windows, ajoutez le repertoire du wrapper a votre variable d'environnement PATH utilisateur.

Astuce
Node.js doit etre installe sur votre systeme pour executer le CLI. Si vous ne l'avez pas, telechargez-le depuis nodejs.org.
