Installazione manuale e configurazione PATH per Bobcorn CLI.

Configurazione manuale

Se preferisci non usare il pannello Impostazioni, puoi configurare il CLI manualmente.

Opzione A: Auto-installazione dal binario CLI

Individua il binario CLI all'interno del bundle dell'applicazione Bobcorn ed esegui direttamente il comando install:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

Questo registra automaticamente il comando bobcorn nel tuo PATH.

Opzione B: Aggiungere al PATH manualmente

Crea uno script wrapper che invoca il binario CLI con Node.js:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

Assicurati che la directory del wrapper sia nel tuo PATH. Su macOS/Linux, aggiungi ~/.local/bin al tuo profilo shell. Su Windows, aggiungi la directory del wrapper alla variabile d'ambiente PATH dell'utente.

Suggerimento
Node.js deve essere installato sul tuo sistema per eseguire il CLI. Se non lo hai, scaricalo da nodejs.org.
