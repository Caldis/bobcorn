Ручная настройка CLI

Ручная установка и настройка PATH для Bobcorn CLI.

Ручная настройка

Если вы предпочитаете не использовать панель настроек, CLI можно настроить вручную.

Вариант A: Самоустановка из бинарного файла CLI

Найдите бинарный файл CLI в пакете приложения Bobcorn и выполните команду install напрямую:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

Это автоматически регистрирует команду bobcorn в вашем PATH.

Вариант B: Добавить в PATH вручную

Создайте скрипт-обёртку, который вызывает бинарный файл CLI через Node.js:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

Убедитесь, что каталог обёртки находится в вашем PATH. На macOS/Linux добавьте ~/.local/bin в профиль оболочки. На Windows добавьте каталог обёртки в пользовательскую переменную среды PATH.

Совет
Для запуска CLI в системе должен быть установлен Node.js. Если у вас его нет, скачайте с nodejs.org.
