CLI Manuel Kurulum

Bobcorn CLI icin manuel kurulum ve PATH yapilandirmasi.

Manuel kurulum

Ayarlar panelini kullanmak istemiyorsaniz, CLI'yi manuel olarak kurabilirsiniz.

Secenek A: CLI ikili dosyasindan kendi kendine kurulum

Bobcorn uygulama paketi icindeki CLI ikili dosyasini bulun ve install komutunu dogrudan calistirin:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

Bu, bobcorn komutunu otomatik olarak PATH'inize kaydeder.

Secenek B: PATH'e manuel olarak ekleme

CLI ikili dosyasini Node.js ile cagiran bir wrapper betigi olusturun:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

Wrapper dizininin PATH'inizde oldugundan emin olun. macOS/Linux'ta kabuk profilinize ~/.local/bin ekleyin. Windows'ta wrapper dizinini kullanici PATH ortam degiskenine ekleyin.

Ipucu
CLI'yi calistirmak icin sisteminizde Node.js yuklu olmalidir. Yuklu degilse nodejs.org adresinden indirin.
