Pengaturan CLI Manual

Instalasi manual dan konfigurasi PATH untuk Bobcorn CLI.

Pengaturan manual

Jika Anda lebih suka tidak menggunakan panel Pengaturan, Anda dapat mengatur CLI secara manual.

Opsi A: Instalasi mandiri dari biner CLI

Temukan biner CLI di dalam bundel aplikasi Bobcorn dan jalankan perintah install secara langsung:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

Ini secara otomatis mendaftarkan perintah bobcorn ke PATH Anda.

Opsi B: Tambahkan ke PATH secara manual

Buat skrip wrapper yang memanggil biner CLI dengan Node.js:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

Pastikan direktori wrapper ada di PATH Anda. Di macOS/Linux, tambahkan ~/.local/bin ke profil shell Anda. Di Windows, tambahkan direktori wrapper ke variabel lingkungan PATH pengguna.

Tips
Node.js harus terinstal di sistem Anda untuk menjalankan CLI. Jika belum memilikinya, unduh dari nodejs.org.
