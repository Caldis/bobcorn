Bobcorn CLI의 수동 설치 및 PATH 구성.

수동 설정

설정 패널을 사용하지 않으려면 CLI를 수동으로 설정할 수 있습니다.

옵션 A: CLI 바이너리에서 자체 설치

Bobcorn 애플리케이션 번들 내부의 CLI 바이너리를 찾아 install 명령을 직접 실행합니다:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

이렇게 하면 bobcorn 명령이 자동으로 PATH에 등록됩니다.

옵션 B: PATH에 수동으로 추가

Node.js로 CLI 바이너리를 호출하는 래퍼 스크립트를 만듭니다:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

래퍼 디렉토리가 PATH에 포함되어 있는지 확인하세요. macOS/Linux에서는 셸 프로필에 ~/.local/bin을 추가하세요. Windows에서는 사용자 PATH 환경 변수에 래퍼 디렉토리를 추가하세요.

팁
CLI를 실행하려면 시스템에 Node.js가 설치되어 있어야 합니다. 설치되어 있지 않다면 nodejs.org에서 다운로드하세요.
