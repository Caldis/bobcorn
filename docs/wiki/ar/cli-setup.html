إعداد CLI يدوي

التثبيت اليدوي وتكوين PATH لـ Bobcorn CLI.

الإعداد اليدوي

إذا كنت تفضل عدم استخدام لوحة الإعدادات، يمكنك إعداد CLI يدويًا.

الخيار أ: التثبيت الذاتي من ملف CLI الثنائي

حدد موقع ملف CLI الثنائي داخل حزمة تطبيق Bobcorn وقم بتشغيل أمر install مباشرة:

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

يؤدي هذا إلى تسجيل أمر bobcorn في PATH تلقائيًا.

الخيار ب: الإضافة إلى PATH يدويًا

أنشئ سكريبت مُغلِّف يستدعي ملف CLI الثنائي باستخدام Node.js:

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

تأكد من أن مجلد المُغلِّف موجود في PATH. على macOS/Linux، أضف ~/.local/bin إلى ملف تعريف الصدفة. على Windows، أضف مجلد المُغلِّف إلى متغير بيئة PATH الخاص بالمستخدم.

تلميح
يجب أن يكون Node.js مثبتًا على نظامك لتشغيل CLI. إذا لم يكن لديك، قم بتنزيله من nodejs.org.
