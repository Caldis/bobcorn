CLI 手動セットアップ

Bobcorn CLI の手動インストールと PATH 設定。

手動セットアップ

設定パネルを使用したくない場合は、CLI を手動でセットアップできます。

オプション A: CLI バイナリからセルフインストール

Bobcorn アプリケーションバンドル内の CLI バイナリを見つけて、install コマンドを直接実行します：

macOS 
node /Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs install
Windows (PowerShell) 
node "$env:LOCALAPPDATA\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" install

これにより、bobcorn コマンドが自動的に PATH に登録されます。

オプション B: PATH に手動で追加

Node.js で CLI バイナリを呼び出すラッパースクリプトを作成します：

macOS / Linux (~/.local/bin/bobcorn) 
#!/bin/sh
exec node "/Applications/Bobcorn.app/Contents/Resources/app.asar.unpacked/out/cli/index.cjs" "$@"
Windows (%LOCALAPPDATA%\Bobcorn\cli\bobcorn.cmd) 
@echo off
node "%LOCALAPPDATA%\Programs\Bobcorn\resources\app.asar.unpacked\out\cli\index.cjs" %*

ラッパーのディレクトリが PATH に含まれていることを確認してください。macOS/Linux では、シェルプロファイルに ~/.local/bin を追加します。Windows では、ユーザー PATH 環境変数にラッパーディレクトリを追加します。

ヒント
CLI を実行するには、システムに Node.js がインストールされている必要があります。まだインストールしていない場合は、nodejs.org からダウンロードしてください。
