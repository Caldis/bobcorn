Icon Management

Select, search, and organize at scale

Import thousands of SVG icons in one drag. Multi-select for batch operations — move to groups, copy, favorite, delete, or export as SVG. Search and filter instantly across your entire library. Dark mode included.

Font Generation

Every format, one click

Generate production-ready icon fonts from your library. Choose groups, pick formats, set an output path — Bobcorn handles the rest. CSS, JavaScript, and HTML demo pages are generated alongside your fonts.

Organization

Groups that keep icons in order

Move icons between groups in seconds. Create, reorder, and nest groups to match your project structure. Save everything as a .icp project file and pick up exactly where you left off.
