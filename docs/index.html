Icon Management & Font Generation Features

Inline Color Editor

Pick colors with hex picker, eyedropper, or type any CSS format. Preview SVG icon changes instantly in the grid.

Scales to Thousands

Manage icon libraries with 3,000+ icons smoothly. Content-visibility, chunked rendering, and tiered sync keep everything fast.

One-Click Font Generation

Generate icon fonts in SVG, TTF, WOFF, WOFF2, EOT plus CSS, JS, and HTML demo — all at once with real-time progress.

Icon Organization

Organize icons into drag-to-reorder groups. Copy, move, recycle. Search and filter across your entire icon library.

Dark Mode

Full dark mode with one-click toggle. Every component adapts seamlessly, persisted across sessions.

Project Files

Save and reload .icp project files. Pick up where you left off. Exports include the project file automatically.
